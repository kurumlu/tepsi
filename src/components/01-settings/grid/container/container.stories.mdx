# Container

## Available props:

| Name        | Type            | Required | Default value |
|-------------|-----------------|----------|---------------|
| `gutter`    | *boolean*       |          | *true*        |
| `type`      | *ContainerType* |          |               |
| `className` | *string*        |          |               |

```javascript
enum ContainerType {
  FULL_WIDTH = 'grid-full-width-no-gutters',
  FLUID = 'fluid-container',
}
```
