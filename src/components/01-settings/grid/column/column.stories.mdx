# Column

## Available props:

| Name              | Type                        | Required | Default value |
|-------------------|-----------------------------|----------|---------------|
| `parent`          | *boolean*                   |          |               |
| `gutter`          | *boolean*                   |          | *true*        |
| `gutterLeft`      | *boolean*                   |          |               |
| `gutterRight`     | *boolean*                   |          |               |
| `gutterBottom`    | *boolean*                   |          |               |
| `gutterTop`       | *boolean*                   |          |               |
| `noGutterLeft`    | *boolean*                   |          |               |
| `noGutterRight`   | *boolean*                   |          |               |
| `grid`            | *number or ColumnGridProps* |          | *12*          |
| `mobile`          | *number or ColumnGridProps* |          |               |
| `mobileLarge`     | *number or ColumnGridProps* |          |               |
| `tabletSmall`     | *number or ColumnGridProps* |          |               |
| `tablet`          | *number or ColumnGridProps* |          |               |
| `desktop`         | *number or ColumnGridProps* |          |               |
| `desktopSpecific` | *number or ColumnGridProps* |          |               |
| `desktopLarge`    | *number or ColumnGridProps* |          |               |
| `className`       | *string*                    |          |               |
| `dataQA`          | *string*                    |          |               |

```javascript
interface ColumnGridProps {
  span: number;
  pull?: number;
  push?: number;
};
```
