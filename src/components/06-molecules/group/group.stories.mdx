# Group

Use to group a bunch of items together and make them wrap responsively.

## Available props:
<ArgsTable of={Group} />

#### GroupAlign
| Key             |
|-----------------|
| `LEFT`          |
| `CENTER`        |
| `RIGHT`         |
| `SPACE_BETWEEN` |

#### GroupVerticalAlign
| Key      |
|----------|
| `TOP`    |
| `CENTER` |
| `BOTTOM` |
| `BASELINE` |

#### GroupGutter
| Key      |
|----------|
| `NONE`   |
| `XSMALL` |
| `SMALL`  |
| `MEDIUM` |
| `LARGE`  |
| `XLARGE` |

#### GroupWrap
| Key       |
|-----------|
| `WRAP`    |
| `NOWRAP`  |
| `REVERSE` |
