# Media query wrapper

Use this component to show content on a specific view size.

## Available props
| Name    | Type                     | Required | Default value |
|---------|--------------------------|----------|---------------|
| `query` | *MediaQueryWrapperQuery* | *false*  | *MOBILE*      |

#### MediaQueryWrapperQuery
| Key             |
|-----------------|
| *MOBILE*        |
| *DESKTOP*       |
| *TABLET*        |
| *TABLET-SMALL * |
