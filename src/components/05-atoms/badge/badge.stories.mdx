# Badge

> Badges show the user a count of something.
> They are typically placed next or on top of another element, and attract attention.

<Canvas>
  <Story name="Default">
    <Badge count={1} />
  </Story>
</Canvas>

---

## Available props

| Name        | Type         | Required    | Default value |
|-------------|--------------|-------------|---------------|
| `count`     | *number*     | <Required/> |               |
| `max`       | *number*     |             | *99*          |
| `color`     | *BadgeColor* |             |               |
| `size`      | *BadgeSize*  |             |               |
| `className` | *string*     |             |               |
| `dataQA`    | *string*     |             |               |

---

### Count

Use the `count` prop to set the value of the badge.

```javascript
<Badge count={...} />
```

<Canvas>
  <Story name="Count">
    <Badge count={24} />
  </Story>
</Canvas>

---

### Max

Use the `max` prop to set a maximum limit of the `count`.

```javascript
<Badge count={...} max={...} />
```

<Canvas>
  <Story name="Max">
    <Badge count={300} max={9} />
  </Story>
</Canvas>

---

### Color

Use the `color` prop to set a color.

| BadgeColor  |
|-------------|
| `SECONDARY` |
| `SUBTLE`    |

```javascript
<Badge color={...} />
```

#### Secondary

<Canvas>
  <Story name="Color - secondary">
    <Badge count={1} color={BadgeColor.SECONDARY} />
  </Story>
</Canvas>

#### Subtle

<Canvas>
  <Story name="Color - subtle">
    <Badge count={1} color={BadgeColor.SUBTLE} />
  </Story>
</Canvas>

---

### Size

Use the `size` prop to set a size.

| BadgeSize |
|-----------|
| `LARGE`   |

```javascript
<Badge size={...} />
```

#### Large

```javascript
<Badge size={BadgeSize.LARGE} />
```

<Canvas>
  <Story name="Size - large">
    <Badge count={1} size={BadgeSize.LARGE} />
  </Story>
</Canvas>
