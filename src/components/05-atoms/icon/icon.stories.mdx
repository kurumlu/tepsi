## Available props

<ArgsTable for={Icon} />

### Name

Use the `name` prop to choose an icon. Uses a component from `IconName`.

```javascript
<Icon name={...} />
```

### Color

Use the `color` prop to change the color of the icon. Uses the `IconColor` enum.

```javascript
<Icon color={...} />
```

Use the `size` prop to change the size of the icon. Uses the `IconSize` enum.

```javascript
<Icon size={...} />
```

### Animation

Use the `animation` prop to set the special type of the animation for icon. Uses the `IconAnimation` enum.

```javascript
<Icon animation={...} />
```
