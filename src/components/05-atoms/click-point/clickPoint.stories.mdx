# Click point

> Click points can be placed within a svg element with coordinates and allow users to click on them.

---

## Available props

| Name       | Type                       | Required    | Default value |
|------------|----------------------------|-------------|---------------|
| `x`        | *string* or *number*       |             | *'50%'*       |
| `y`        | *string* or *number*       |             | *'50%'*       |
| `onClick`  | *(e?: MouseEvent) => void* | <Required/> |               |
| `isActive` | *boolean*                  |             |               |

---

### Position

Use the `x` and `y` props to set the position of the click point. This position is used as the center of the click point.

```javascript
<ClickPoint x={...} y={...} />
```

These values can be a *string* or a *number*. Meaning that you can pass percentages or absolute pixel values.

```javascript
<ClickPoint x="20%" y="50%" />
```

```javascript
<ClickPoint x={15} y={40} />
```

---

### Click event

Use the `onClick` prop to pass a function that gets called when the click point is clicked.

```javascript
<ClickPoint onClick={...} />
```

---

### Active state

Use the `isActive` prop to pass a boolean to indicate wether to show the click point in an active state or not.

```javascript
<ClickPoint isActive />
```
