# Tag

> Tags show the user information about something.
> They are typically placed next or on top of another element, and attract attention.

<Canvas>
  <Story name="Default">
    <Tag>
      Default
    </Tag>
  </Story>
</Canvas>

## Available props

| Name        | Type       | Required | Default value |
|-------------|------------|----------|---------------|
| `color`     | *TagColor* |          |               |
| `dataQA`    | *string*   |          |               |
| `className` | *string*   |          |               |

---

### Color

Use the `color` prop to set the color of the tag. Uses the `TagColor` enum.

| TagColor      |
|---------------|
| `PRIMARY`     |
| `SECONDARY`   |
| `TERTIARY`    |
| `QUATERNARY`  |
| `QUINARY`     |
| `SENARY`      |
| `SEPTENARY`   |
| `GENERAL`     |
| `DISCOUNT`    |
| `SUSTAINABLE` |

```javascript
<Tag color={...}>...</Tag>
```
