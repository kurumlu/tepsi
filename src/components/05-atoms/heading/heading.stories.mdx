### Priority

The `priority` prop relates to the level of the h-tag that will be used. And does not relate to visual changes.
Uses the `HeadingPriority` enum.
By default this is set to render an `<h2>` tag.

| HeadingPriority |
|-----------------|
| `H1`            |
| `H2`            |
| `H3`            |
| `H4`            |
| `H5`            |

```javascript
<Heading priority={...}>...</Heading>
```

#### H1

```javascript
<Heading priority={HeadingPriority.H1}>...</Heading>
```
