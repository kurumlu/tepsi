## Available props

| Name     | Type          | Required | Default value       |
|----------|---------------|----------|---------------------|
| `size`   | *SpinnerSize* |          | *SpinnerSize.LARGE* |


---

### Size

Use the `size` prop to set the size of the spinner. Uses the `SpinnerSize` enum. Defaults to Large (48px).

| SpinnerSize  |
|--------------|
| `LARGE`      |
| `SMALL`      |
