### Variant

Use the `variant` prop to set the variant of the button.

| Variant     |
| ----------- |
| `PRIMARY`   |
| `SECONDARY` |
| `TERTIARY`  |

```javascript
<Button variant={...} >...</Button>
```

